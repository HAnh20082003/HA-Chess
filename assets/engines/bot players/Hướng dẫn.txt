* Không được phép xoá file Bot Players.txt vì nó giúp phần mềm đọc được các con bots cho chế độ đấu máy.
* Bạn có thể tự tạo 1 con bot cho riêng mình bằng cách:
     B1) có các file sau:
	- 1 file hình ảnh đuôi .png
	- 1 file txt trong đó ghi các thông tin sau:
		+ Name: tên bot
		+ Elo: elo bot
		+ Level: cấp độ bot (từ 0 đến 20)
     B2) Tạo folder với tên bất kì vd là MyBot và bỏ tất cả các file trên vào trong
     B3) Mở file Bot Players.txt lên thêm 1 dòng là tên của folder vừa tạo vd là MyBot
